# Standards de code

- ESLint/Prettier obligatoires
- Nommer clairement (anglais technique)
- Pas de code mort

## Interdits
- `console.log` en prod
- `any` sans justification
- Skip de tests sans justification

## Documentation obligatoire
- Le **code doit être documenté** (comments/README technique)
- La **documentation fonctionnelle** est obligatoire pour chaque module/feature
